The Books Bench Project
This application catalogs and tracks all of the technical books that Catalyst provides to the developers. The system allows checkout, returns, searches and reservations.
This web application uses a single page application (SPA) model, incorporating existing JavaScript frameworks designed for the purpose.

Front End Technologies

BootStrap CSS
Used for the website layout
Angular JS
Used for data representation
Single Page Web Application
"a web application or web site that fits on a single web page with the goal of providing a more fluid user experience akin to a desktop application."

Backend, Testing and Database Technologies

Java
Used to create backend controllers and DAOs
PosgreSQL
Used as the database for book records
Apache Tomcat
Used as the web application's server
Selenium
Used for web testing.
JMeter
Used for web server response testing.
Jasmine and PhantomJs
Used for angular-JS testing
Jenkins
Used for testing coverage monitoring.
